# PyGameZero.sk
pygame Zero php/html Skulpt Web Server

## how to use
```batch

git clone https://github.com/SMLkaiellis08/PyGameZero.sk.git

cd PyGameZero.sk

:: open folder
start .

:: Start The Webserver
start WebServer_pgzero_start.bat
```
